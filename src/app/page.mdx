## Disclaimer

> This API is **unofficial** and **not affiliated** with AniList, Jikan, or any third-party providers.

## Base URL

All API requests should be made to:  
[https://hakai-api.vercel.app](https://hakai-api.vercel.app)

> **Note:** The API supports only **GET** requests with a default rate limit of **6 requests per second**.

## Prebuilt API

The source code for the prebuilt API is available on GitHub:  
[https://github.com/middlegear/hakai-api](https://github.com/middlegear/hakai-api)
