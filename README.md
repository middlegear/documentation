<h1 align="center">Hakai API - Usage Documentation</h1>

<p align="center">
  A REST API for accessing anime metadata. 
</p>

---

## 🚨 Disclaimer

> This API is **unofficial** and **not affiliated** with Anilist, Jikan, or any third-party providers.  
> It does not host, own, or distribute any content. All data belongs to its respective owners.

---

## ✨ API Features

- 🔍 **Search Anime** by title  
- 📜 **Retrieve Metadata** (episodes, details)  
- 📡 **Fetch Streaming Servers**  
- 🎥 **Get Watch Links** for anime episodes  

🔹 The API supports **caching (Redis & Memory)** and **rate limiting** to enhance performance.

---
